# Funder Discovery Workflow

## Recommended Sequence

1. Run Organization Details.
2. Run Prospect Discovery against that profile.
3. Generate briefs for the strongest prospects.
4. Apply Fit Scoring.
5. Build the pipeline export.

## Evidence Rules

Use the most recent filing year available, but name the year. If filings are stale, say so.

Separate evidence from inference. A grant to a similar grantee is evidence. A foundation website sentence is a signal. A model guess about openness is an inference.

Score unknowns conservatively. Missing data should lower confidence before it raises priority.

Do not recommend outreach without a next action. A pipeline row without an action is research residue, not fundraising work.
